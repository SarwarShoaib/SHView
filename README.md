# SHView
SHView is a view with gone property.

## Features

- Just setGone -- view will disapear.
- Automatically adjust constaraints height.
- Automatically adjust margens.

![Preview](demo.gif)


## Example

```swift
let myView = SHView()
```
add your constraints

```swift
myView.gone = true
```
or
```swift
myView.gone = false
```
